# 云端接口

## 概述

云端接口用于承载开放平台的运营能力和云端数据交互能力，适合在驱动客户端、网页端或业务后台中对接账号、设备、配置同步、运营活动等服务。

当前分类作为云端能力的文档入口，后续具体接口会按照业务模块继续补充。

## 接口范围

- 设备与产品信息的云端查询。
- 驱动客户端配置、方案或用户数据的云端同步。
- 活动、素材、公告等运营内容的读取。
- 与开放平台业务后台相关的数据接口。

## 使用建议

TIP

- 云端接口通常需要服务端地址、鉴权方式和业务权限，请在接入前确认当前项目的开放范围。
- 客户端只应保存必要的会话信息，敏感密钥建议放在服务端处理。
- 与硬件实时通讯相关的能力请优先查看键盘、鼠标和固件升级接口分类。
